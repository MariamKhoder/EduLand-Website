# Eduland - Online Learning Platform

##  Project Overview
Eduland is a comprehensive educational website designed to provide a seamless learning experience for students. This project was developed as a **Collaborative Team Project**, focusing on creating a responsive and user-friendly interface for an online course platform.

##  Team Collaboration
Working in a team allowed us to simulate a real-world development environment. We collaborated on:
- **UI/UX Design:** Crafting a consistent visual identity across all pages.
- **Frontend Development:** Dividing the modules (Home, Courses, Auth, Contact) to ensure efficient progress.
- **Problem Solving:** Conducting code reviews and fixing layout issues collectively.

##  Key Features
- **Home Page:** Engaging hero section with clear Call-to-Actions.
- **Course Catalog:** Filterable course categories (AI, Engineering, Business, etc.).
- **Authentication:** Custom-designed Sign In and Sign Up pages.
- **Contact Us:** Interactive contact form with a modern glassmorphism effect.
- **About Us:** A section dedicated to the team's mission and vision.

##  Tech Stack
- HTML5
- CSS3 (Custom styling & Layouts)
- JavaScript (for interactivity)
- Responsive Design Principles
